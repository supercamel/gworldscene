GWorldScene
===========

GWorldScene is a GTK 3 and GTK 4 ``GtkGLArea`` scene widget for rendering
geospatial terrain, map imagery, an Earth-scale globe view, and scene graph
objects placed with latitude, longitude, and altitude.

The public API is available as C libraries, GObject Introspection typelibs, and
generated bindings such as Vala and SQGI. The shared core library contains the
scene data and geospatial math, while backend-specific libraries expose the GTK
widget for GTK 3 or GTK 4 applications.
