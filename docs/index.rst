GWorldScene
===========

GWorldScene is a GTK 4 ``GtkGLArea`` widget for rendering geospatial terrain,
map imagery, an Earth-scale globe view, and scene graph objects placed with
latitude, longitude, and altitude.

The public API is available as a C library, a GObject Introspection typelib,
and generated bindings such as Vala and SQGI.
